在Ubuntu上安装java环境

2021年07月22日 留言

JDK的安装

1. 下载安装包 提取码：nmrf

2. 创建目录存放jvm

3. 将下载好的压缩包放在系统中，并在终端cd，我放在了主目录的下载下面

4. 配置环境变量

5. 进入了vim编辑器，可见光标在最开始，敲击i进入编辑模式，粘贴shift+ctrl+v以下代码到最开始

6. esc退出编辑模式，键入：wq保存退出文件

7. 让配置立即生效

8. 查看java版本
