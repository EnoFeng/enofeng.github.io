问题：Vue ui打开控制面板的时候，卡在这个界面

解决

在你cmd打开的默认目录下输入explorer %cd%，打开.vuerc文件（记事本或者文本编辑器都可以），把第一个使用淘宝镜像的设置改为true

问题2

在经历上面的修改后的若干天后，又出现了

解决2

和上一个解决相同操作，改为false，或者换个网络
